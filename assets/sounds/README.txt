Ця папка містить звукові файли для додатку Gurdjieff Stop.

Необхідні файли (формат .mp3 або .ogg):
- bell.mp3          — дзвін
- tibetan_bowl.mp3  — тибетська чаша
- soft_chime.mp3    — м'який дзвінок
- gong.mp3          — гонг
- wind_bell.mp3     — вітряний дзвінок

Безкоштовні звуки можна знайти на:
- https://freesound.org (пошук: "bell", "tibetan bowl", "chime")
- https://pixabay.com/music/

Завантажте файли з відповідними іменами у цю папку,
потім виконайте: flutter pub get && flutter build apk
